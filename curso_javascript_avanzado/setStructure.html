Estructura Set

Estructura de datos compuesta por un conjunto de valores únicos.

Características

  • No permite datos repetidos.
  • Es posible crear un set a partir de cualquier objeto iterable de Javascript.
  • Útil para obtener valores no repetidos de una lista.
  • Al añadir nuevos valores, si son objetos, el set comprueba sus referencias.
  • Es posible iterar los sets utilizando keys(), values(), entries().
  • Puede crearse un array a partir de un set gracias al uso de destructuring.
  • Si creamos un set de objetos, lo que se comprueba son las referencias, no los valores.

Métodos

  • add(): añade el elemento que le pasamos. No lo añade si está repetido.
  • has(): comprueba si el elemento que le hemos pasado, se encuentra en el set.
  • delete(): busca el valor que le pasamos y lo borra del set.
