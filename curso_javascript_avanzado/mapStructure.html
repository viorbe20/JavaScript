Estructura Map

Estructura de datos compuesta por registro formados por una clave y un valor.

Características

  • Es posible crear un set a partir de cualquier objeto iterable de Javascript.
  • Sus claves pueden ser objetos, funciones o cualquier tipo primitivo.
  • Para el tamaña utilizamos función size().
  • Es iterable.

Métodos

  • get(): le pasamos la clave y devuelve el valor.
  • set(): le pasamos la clave y establece el valor.
  • delete(): busca el valor que le pasamos y lo borra del map.
